# wdroid 🚀

Automate Waydroid like a pro.

CLI profissional para gerenciar Waydroid no Debian com foco em automação, estabilidade e extensibilidade.

---

## ✨ Features

- State machine inteligente
- Retry com backoff
- Logs estruturados
- Sistema de plugins
- Backup automático
- Doctor com auto-fix

---

## 📦 Instalação

```bash
git clone https://github.com/seu-usuario/wdroid
cd wdroid
make install
